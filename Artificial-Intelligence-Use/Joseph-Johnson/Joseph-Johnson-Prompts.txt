● That's your issue. It's not quite normal — here's what's happening:

  Your project has src/test/java and src/main/java in the repo (the files are there), but Eclipse is not showing the Maven source folder layout because the .classpath file is missing.
                                                                                                                                                                                                                                                                                                                               Eclipse needs two files to understand a Maven project properly:                                                                                                                                                                                                                                                              - .project — present (and correct, it has the Maven nature)                                                                                                                                                                                                                                                                  - .classpath — missing — this tells Eclipse where src/main/java, src/test/java, etc. are                                                                                                                                                                                                                                   
  Without .classpath, Eclipse falls back to just showing raw packages, which is what you're seeing.

  How to fix it:

  The cleanest way is to let Eclipse regenerate both files via Maven:

  1. Right-click the project in Eclipse's Package Explorer
  2. Go to Maven > Update Project... (or press Alt+F5)
  3. Check "Force Update of Snapshots/Releases"
  4. Click OK

  Eclipse's M2E plugin will re-read pom.xml and regenerate the .classpath, which will properly configure src/main/java and src/test/java as source folders.

  After that you should see the standard Maven layout with the separate source folders.
